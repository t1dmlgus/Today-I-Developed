# 영화 예매 시스템(feat.롯데시네마)

<br>

- 책임 주도 개발로 도메인 간 협력을 통해 기능 구현
- RESTful API 설계
- ErrorType 을 활용한 예외처리
- 인터셉터를 통해 인증, 인가 처리


### 기술 스택

* Java11
* Spring Boot 2.7.2
* Spring Data JPA
* querydsl 5.0
* Mysql, H2
* Junit5
* docker


### 요구사항

- 영화 예매 시스템을 구현한다.
  - 예매 시스템은 영화를 예매하는 것이 아닌, 해당 영화의 상영시간을 예매하는 것이다.**

- 영화관을 등록한다.(feat. 상영관)
  

- 매일 상영시간표를 초기화 한다.(시간 미정)


### 도메인 개념 도출

- 영화관(Cinema)
- 상영관(Theater)
- 영화(Movie)
- 상영(Screen)
- 예약(Reservation)
